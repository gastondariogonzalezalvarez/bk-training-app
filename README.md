# BK Training App 🍔📖

Aplicación web progresiva (PWA) de entrenamiento interactivo para el personal de **Burger King**, diseñada para facilitar el estudio del manual operativo 2025 de forma dinámica y eficiente.

## ✨ Características Principales

- **📖 100 Preguntas Actualizadas**: Base de datos completa basada en los estándares del manual 2025.
- **📍 Referencias de Página**: Cada explicación incluye la página exacta del manual (ej: Pág. 116) para facilitar la consulta.
- **📈 KPIs por Categoría**: Seguimiento en tiempo real de tu rendimiento en áreas críticas como Seguridad Alimentaria, Higiene, Atención al Cliente y más.
- **🔄 Lógica de No Repetición**: No verás la misma pregunta dentro de una sesión hasta que hayas agotado todas las de la categoría seleccionada.
- **📱 PWA (Progressive Web App)**: Instalable en dispositivos Android e iOS sin necesidad de tiendas de aplicaciones. Funciona a pantalla completa.
- **📶 Soporte Offline**: Una vez cargada, la aplicación funciona sin conexión a internet.
- **💾 Persistente**: Guarda tus récords y nombres de usuario localmente en el dispositivo.

## 🚀 Instalación en Móvil (Android/Chrome)

1. Abre el enlace de la aplicación en Chrome.
2. Toca el menú de **tres puntos** (arriba a la derecha).
3. Selecciona **"Instalar aplicación"** o **"Añadir a la pantalla de inicio"**.
4. ¡Listo! El icono de BK aparecerá en tu escritorio y funcionará como una app nativa.

## 🛠️ Tecnologías

- **Vanilla JS**: Lógica pura y rápida.
- **CSS3**: Diseño responsive moderno y animaciones fluidas.
- **HTML5 Semantic**: Estructura accesible.
- **Service Workers**: Para el funcionamiento offline.
